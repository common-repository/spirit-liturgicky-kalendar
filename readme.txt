Spirit Liturgický kalendár 

== SK ==
Liturgický kalendár podľa stránky lc.kbs.sk. 
Obsahuje widget a gutenberg block, ktoré zobrazia aktuálny aleluja verš a názov dnešného dňa spolu s odkazom na oficiálnu stránku liturgického kalendára. 

== EN ==
This plugin displays daily readings from Holy Bible. 
It fetches data from Slovak web site of Catholic church in Slovakia (lc.kbs.sk).
Therefore it is localized only in Slovak language.

== Installation ==

1. Plugin nainštalujete priamo cez Wordpress -> Administrácia -> Pluginy -> Pridať nový -> Hľadajte Liturgický kalendár
2. Prípadne stiahnite si .zip súbor a nahrajte ho cez možnosť "Nahrať plugin" v administrácii.
